# CRUD Java Mangas

Projeto simples em Java que implementa um CRUD (Create, Read, Update e Delete) de mangás.

A ideia do projeto surgiu para me ajudar a organizar os mangás e volumes que já li, e ao mesmo tempo praticar manipulação de dados e estruturação de um sistema em Java.

No futuro, pretendo evoluir o projeto para uma versão web.

---

## 📌 Funcionalidades

O sistema permite:

* Cadastrar um mangá
* Listar mangás cadastrados
* Atualizar informações de um mangá
* Remover um mangá

Os dados são armazenados em um banco local (`mangas.db`).

---

## 🛠 Tecnologias utilizadas

* Java
* Banco de dados local
* IntelliJ e VS code

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/isaindex/CRUD-java-mangas.git
```

2. Abra o projeto na sua IDE.

3. Execute a classe principal (ex: `Main`).

Ou pelo terminal:

```bash
javac *.java
java Main
```
